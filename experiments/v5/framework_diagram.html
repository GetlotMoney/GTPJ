GTPJ-v5 框架图

v5 激活 TRIAL-003：把 all_text_cond [B,C,768] 直接送入 BVSA，本图是版本级主线视图；trial 级细节仍以 experiments/module_trials/IDEA-0002_fae_memory_jepa/TRIAL-003_conditional_bvsa_text/framework_diagram.md 为准。

输入

图像 batch，GPT 类别文本描述，CUB 类别集合。

↓

Frozen CLIP

clip_features [B,577,768]，拆成 CLS token 和 patch tokens。

PSE / CLIP-A-self

生成 all_text [C,768]；v5 使用 pse_outer_ratio=0.65。

↓

ICSA

用图像 CLS token 注入条件语义，得到 all_text_cond [B,C,768]。

↓

全局分数

S_global [B,C]，由图像全局特征和条件文本原型计算。

FGVD

选择 K=32 个局部 patch 并编码为几何感知视觉记忆。

↓

BVSA

v5 关键变化：BVSA 接收 all_text_cond，而不是共享 all_text。

↓

局部分数

V2S / S2V 解码器生成 S_local [B,C]。

融合 logits

S_final = S_global + 0.2 * S_local，最终形状 [B,C]。

↓

GZSL 评估

U / S / H / ZS；seen/unseen split、label mapping、class order 不变。

↓

辅助训练

SGMP 读取 FGVD memory 和 conditional text，产生 L_mpp / L_neg。

关键差异

bvsa_text_mode=conditional 让 ICSA 影响 BVSA local branch；关闭到 adapted 时回到共享文本路径。

训练轮数

epochs=30 是历史字段；有效计划来自 lr_stages，即 20 + 20 + 10 = 50 轮。

证据边界

v5 是 owner-activated mainline；repeat mean H=74.44，不能说成强于 v3/CONFIRM-001 的 confirmed baseline。
